# Как скачать готовый .exe файл для Windows

## Способ 1: Через GitHub Actions (без установки Python)

1. Перейдите на страницу: https://github.com/VToreZ/htm-processor/actions

2. Найдите последний успешный запуск **"Build Windows EXE"** (со значком ✓)

3. Прокрутите вниз до раздела **Artifacts**

4. Скачайте архив **HTM_Processor_Windows**

5. Распакуйте архив - внутри будет файл **HTM_Processor.exe**

6. Запустите **HTM_Processor.exe** - программа готова к работе!

**Примечание:** .exe файл собран специально для Windows, не требует установки Python.

---

## Способ 2: Скачать архив с исходниками (требуется Python)

1. Скачайте последнюю версию: https://github.com/VToreZ/htm-processor/archive/refs/heads/master.zip

2. Распакуйте архив

3. Следуйте инструкциям из файла **INSTALL.txt**
